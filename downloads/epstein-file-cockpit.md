# Epstein Actual Files Cockpit

Updated: 2026-06-25

## Boundary
A document route is a starting point, not a conclusion. Readers should open the file, identify the record type, check the date, read the context, and classify what it supports before drawing a conclusion.

## Open These File Doors First
- DOJ Epstein Disclosures: https://www.justice.gov/epstein/doj-disclosures — Open the official DOJ disclosure route for government-published Epstein material, redactions, indexes, and release notes.
- House Oversight Epstein Records: https://oversight.house.gov/?s=Epstein — Open the congressional oversight lane for statements, witness testimony, release announcements, subpoenas, and committee framing.
- CourtListener Epstein Dockets: https://www.courtlistener.com/?q=Epstein&type=r&order_by=score%20desc — Search court records, filings, motions, civil suits, rulings, habeas arguments, Maxwell-related litigation, and related docket material.
- Jmail Epstein Email Search: https://jmail.world — Search the email-style archive by name, topic, sender, recipient, and date. Treat search results as document leads and verify context.
- WikiLeaks Search: Epstein: https://search.wikileaks.org/?query=Epstein — Search public archive material for Epstein-related references. Treat results as leads and corroborate before using them as claims.
- FBI Vault Search: https://vault.fbi.gov/ — Search the FBI public vault for released files, historic records, FOIA material, and related investigative records.
- CIA Reading Room: https://www.cia.gov/readingroom/ — Search declassified CIA reading-room material where intelligence-history context is relevant.
- National Archives Catalog: https://catalog.archives.gov/ — Search U.S. archival records, agency material, photographs, legal records, and historical files.
- SEC EDGAR: https://www.sec.gov/edgar/search/ — Search corporate filings, directors, funds, entities, securities records, and financial relationships that may help map money or institutional access.
- FARA Records: https://efile.fara.gov/ords/fara/f?p=1381:1 — Search foreign-agent registration records, lobbying, representation, and influence disclosures.
- OpenSecrets: https://www.opensecrets.org/ — Search campaign finance, donors, PACs, lobbying, and political-money patterns.
- ICIJ Offshore Leaks Database: https://offshoreleaks.icij.org/ — Search offshore entities, intermediaries, and company links. Treat results as leads requiring context and verification.

## How To Use The Cockpit
- Start with the official disclosure or court record where possible.
- Search names across email, court, oversight, and financial routes separately.
- Record the document type before interpreting the meaning.
- Separate contact, claim, sworn testimony, settlement, and conviction.
- Use repeated appearance across independent routes as a stronger network signal.
- Never turn a search result into a criminal finding without court-grade support.
