Signal Classes

Signal Class

Public record

Court, law, official release, hearing, filing, contract, budget, treaty, or documented source item.

Signal Class

Reported record

Dated reporting that points to documents, named sources, or official statements.

Signal Class

Pattern

Multiple source-linked events showing a repeated institutional direction.

Signal Class

Convergence

Two or more systems connect, such as identity plus money, health, speech, travel, policing, AI, or energy.

Signal Class

Hypothesis

A structured question built from records, not a verdict.

Signal Class

Speculation

Motif, internet claim, symbolic reading, or open-source lead requiring stronger evidence.

Signal Class

Debunk lane

Claim-history and counter-source tracking for claims with weak, false, or unsupported source chains.

Operating Rules

  • Expose wrongdoing, not innocent people.
  • Source first, claim second.
  • Every signal needs actor, date, jurisdiction, source class, evidence level, and boundary.
  • Separate fact, allegation, pattern, hypothesis, speculation, and debunk lane.
  • When systems connect, show the convergence link instead of making unsupported leaps.
  • Use safe wording publicly: institutional convergence, policy drift, power concentration, mandate lane, source gap, open question.
  • Expose patterns hard, but do not weaken the site with unsupported legal claims.
  • Never publish private victim identities, illegal material, doxxing, graphic exploitation, or direct accusations without source-grade evidence.
